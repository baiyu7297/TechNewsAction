## 升级指南

### 从 v1.0.0 升级到 v2.0.0

如果你已经配置了 v1.0.0，升级到 v2.0.0 非常简单：

1. **拉取最新代码**
   ```bash
   git pull origin main
   ```

2. **无需修改配置**
   - GitHub Secrets 保持不变
   - 邮箱配置保持不变
   - 定时任务保持不变

3. **查看新内容**
   - 邮件主题变为 "🤖 AI技术资讯"
   - 内容变为中文 AI 新闻
   - 来源变为机器之心、量子位、雷峰网

4. **测试运行**
   - 在 GitHub Actions 中手动触发
   - 检查邮件内容

就这么简单！所有配置都兼容，只是新闻源和内容发生了变化。
